Работа с редактором NANO
        1. nano file_name - открытие файла
        2. CTRL + O - сохранить файл
        3. CTRL + X - выход с редактора
        4. python3 file_name - просмотреть результат исполняемого файла

    Работа с GIT
        1. Проверка изменений на удаленном сервере и внесение на локальную машину - git pull
        2. Проверка изменений на локальной машине - git status
